# Celebal Data Engineering Assignment 1

This project contains basic data cleaning and data exploration using Python pandas and Jupyter Notebook.

## Tasks Performed

- Loaded and explored the dataset
- Checked and handled missing values
- Filtered and selected data
- Removed duplicate entries
- Created derived columns
- Saved the cleaned dataset as CSV

## Tools Used

- Python
- pandas
- NumPy
- Jupyter Notebook

## Files

- `Celebal_Assignment_1.ipynb` - main notebook
- `final_dataset.csv` - cleaned dataset
